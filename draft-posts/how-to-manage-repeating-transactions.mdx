In this tutorial, we are going to learn how we can manage any repeating transactions you might have added to MoneyCoach.

This short tutorial will show you the correct way to stop a repeating transaction from ever being repeated. Please keep in mind that if you want to stop a repeating transaction from being repeated, you need to edit the main repeating transaction, the one that has the "R" icon next to it.
